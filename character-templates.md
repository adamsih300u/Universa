## How to Use

1. **For Major Characters**: Create individual files like `derek-thomas.md`
2. **For Supporting Cast**: Create one file like `supporting-cast.md` 
3. **Use the frontmatter**: Set `type: characters` to activate the Character Development Chain
4. **Reference other characters**: Use `ref_character_name: path/to/file.md` to cross-reference

The AI will now see all your character content and help you develop, expand, and maintain consistency across your series!
